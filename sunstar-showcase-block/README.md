# Sunstar Showcase Block

A custom WordPress Gutenberg block plugin inspired by modern corporate design patterns (like Sun Asterisk).

## Features

- **Hero Section Block**: Beautiful hero banner with title, subtitle, and CTA button
- **Customizable Colors**: Background and text color controls
- **Responsive Design**: Works great on desktop, tablet, and mobile
- **Easy to Use**: Simple block interface with inspector controls

## Installation

1. Place this folder in `wp-content/plugins/`
2. Activate the plugin from WordPress admin
3. Add the "Sunstar Hero Section" block to your pages/posts

## Block Options

- **Title**: Main headline text
- **Subtitle**: Subheading text
- **Background Color**: Hero section background
- **Text Color**: Headline and subtitle color
- **Button Label**: Call-to-action button text
- **Button Link**: Button target URL

## Development

### Build from source

```bash
npm install
npm run build
```

### Development mode with watch

```bash
npm start
```

### Linting

```bash
npm run lint:js
```
